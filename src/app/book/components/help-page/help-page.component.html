📘 Help Page – Book Store

Learn how to use each feature of the site step by step.

🏠 1. Home Page – View Book List

• Click the Home link in the navigation bar.

• Browse the list of books.

• Click the Details button to view full book info.

➕ 2. Add a New Book

• Click the Add Book link in the navigation bar.

• Fill in the form fields like Title, Authors, etc.

• Click the Submit button.

• Your book will appear in the book list.

✏️ 3. Edit a Book

• Click Details on the book you want to edit.

• Click the Edit button.

• Update the fields and click Save.

🗑️ 4. Delete a Book

• Open the book’s Details page.

• Click the Delete button.

• Confirm when prompted.

🔐 5. Login / Logout

• Click Login in the top-right navbar.

• After logging in, you can Add/Edit/Delete books.

• To log out, click Logout.
